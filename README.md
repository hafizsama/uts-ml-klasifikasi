# Klasifikasi Orange vs Grapefruit

## Deskripsi
Uts mengklasifikasikan buah menjadi orange atau grapefruit menggunakan machine learning.

Dataset:
https://www.kaggle.com/datasets/joshmcadams/oranges-vs-grapefruit

## Algoritma yang Digunakan
- Decision Tree
- Naive Bayes
- Support Vector Machine (SVM)

## Tahapan

1. Load dataset menggunakan pandas
2. Exploratory Data Analysis (EDA)
3. Preprocessing (encoding, split data, scaling)
4. Training model:
   - Decision Tree
   - Naive Bayes
   - SVM
5. Evaluasi model menggunakan accuracy, confusion matrix, dan classification report

## Hasil
- Decision Tree: 0.9435
- Naive Bayes: 0.92
- SVM: 0.929

## Kesimpulan

Model Decision Tree memberikan hasil terbaik dibandingkan model lainnya dalam mengklasifikasikan data.
